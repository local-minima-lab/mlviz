# Scenario

You have been tasked to identify **Iris** flowers. There are three different species: Iris setosa, Iris virginica and Iris versicolor. For each sample, you have four features: petal length, petal width, sepal length and sepal width (in centimeters).

You want to create a machine learning model to **classify** these flowers. Your friend, Alex, says: "Why don't you try Decision Trees?"
